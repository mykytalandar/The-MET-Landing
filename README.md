# The MET Landing
Project Description

**The MET Landing** is a modern, responsive landing page dedicated to The Metropolitan Museum of Art.
It provides essential information about the museum — including its visiting hours, current exhibitions, and ongoing programs — in a clean and user-friendly interface.
The goal of this project is to create a visually appealing and informative web page that highlights the museum’s cultural value and improves the visitor experience.

## Technologies Used
**Core**

- HTML5 – Structure and semantic layout

- SCSS (v7.0.3) – Styling with modular and reusable code

- BEM Methodology – Maintainable and scalable CSS architecture

**Development**

- ESLint (v8.45.0) – Code quality and consistency

**Deployment**

- GitHub Pages – Hosting and deployment

## Key Features

- Fully responsive design for all screen sizes (mobile, tablet, desktop)

- Clean and accessible layout based on modern design principles

- Optimized performance with lightweight SCSS and HTML structure

- Easy navigation and clear information hierarchy
